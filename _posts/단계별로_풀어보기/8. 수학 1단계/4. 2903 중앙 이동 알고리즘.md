### **상황**

- #### **문제 내용을 정리하여 작성했다**

1. #### 초기에는 정사각형 모서리에 점이 1개씩 총 4개가 있다.

    + 한 변에 점이 2 개씩 있게 된다.

2. #### 정사각형들을 가로세로로 접어 4개의 작은 정사각형들로 만든다.

3. #### 이 작업을 N 번 반복했을때 점의 갯수를 구한다.

</div>

- N: 정사각형을 접는 횟수

</div>

- 점의 갯수

## **소스 코드**

<div class="block_green">

### **Node**

``` js
// 입력
const fs = require('fs')
const inputArr = fs.readFileSync(0, 'utf-8').toString().split('\n')

// 입력 파싱
const N = +inputArr[0]

// 가로 점 갯수 계산
const width = Math.pow(2, N) + 1

// 점 갯수 계산
const result = width * width

// 출력
process.stdout.write(`${result}`)
```

### **공통**

- #### 리눅스 환경에서 이상한 문자가 추가되어 문자열에 들어가는것을 방지하고자 trim 을 썼다.

- #### 대용량 입출력을 연습하기 위해 process.stdout.write 함수를 쓰는 습관을 들인다.

</div>

<div class="block_grey tp_80">

### **특이점**

- #### width[n] : n 번 접었을때, 한 변에 있는 점의 갯수

- #### width[0] = 2 (초기 조건)

- #### width[n] = 2 * width[n - 1] - 1

- #### 위 점화식을 수식으로 나타내면

    + width[n] = 2*n + 1 이다.

</div>

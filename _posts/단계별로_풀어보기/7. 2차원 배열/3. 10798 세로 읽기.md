### **상황**

- #### 최대 5줄, 줄당 최대 15개의 문자가 행렬 형태로 주어진다.

- #### 이 문자들을 맨 왼쪽부터 세로방향으로 읽을때, 읽는 순서대로 출력한다.

</div>

<div class="block_yellow">

### **입력**

- M : 5줄에 걸쳐서 들어오는 5개의 문자열

</div>

<div class="block_green">

### **출력**

- 세로방향으로 읽었을때, 읽는 문자들을 순서대로 한줄로 출력

</div>

## **소스 코드**

<div class="block_green">

### **Node**

``` js
// 입력
const fs = require('fs')
const inputArr = fs.readFileSync(0, 'utf-8').toString().trim().split('\n')

// 5 * 15 의 공백으로 채워진 배열 생성
const M = Array(15)
  .fill(null)
  .map(() => Array.from({length: 5}).fill(''))

// 배열 M 에 입력을 reverse 하여 입력
inputArr.forEach((row, rowIdx) => {
  row
    .trim()
    .split('')
    .forEach((c, colIdx) => {
      M[colIdx][rowIdx] = c
    })
})

M.forEach(row => {
  process.stdout.write(`${row.join('')}`)
})
```

- #### 대용량 입출력을 연습하기 위해 process.stdout.write 함수를 쓰는 습관을 들인다.

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

M = list(list(input().strip()) for i in range(5))

resultArr = [['' for i in range(5)] for j in range(15)]

for rowIdx in range(5):
  for colIdx, c in enumerate(M[rowIdx]):
    resultArr[colIdx][rowIdx] = c

for row in resultArr:
  output(f"{''.join(row)}")
```

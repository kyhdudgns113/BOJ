## **설명**

- #### Hello World 를 출력하는 문제다.

- #### 입출력 함수 사용이 손에 익게 하기 위해 풀어본다.

</div>

<div class="block_grey">

## **소스 코드**

<div class="block_orange">

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

output("Hello World!")
```

- #### print 함수를 쓰면 코드가 훨씬 간결해진다.

- #### 나중에 대량 입출력을 할 때는 print 함수가 시간초과가 난다.

- #### 미리 연습한다는 느낌으로 사용

</div>

<div class="block_green">

### **Node**

``` js
process.stdout.write("Hello World!")
```

- #### Python 과 마찬가지로 console.log 함수를 쓰면 간결해진다.

- #### 나중에 대량 입출력 사용을 위해 미리 연습한다는 느낌으로 사용

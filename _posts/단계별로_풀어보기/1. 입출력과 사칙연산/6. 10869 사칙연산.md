## **설명**

- 10 미만의 두 자연수 A, B 를 입력받고 A / B 를 출력한다.

- 원래 문제 이름은 A / B 이지만 / 를 파일 이름에 넣을 수 없어서 div 로 대체했다.

</div>

<div class="block_grey">

## **소스 코드**

<div class="block_green_transparent">

### **Node**

``` js
const fs = require('fs')
const input = fs.readFileSync('/dev/stdin').toString().split(' ')
let [A, B] = input
A = parseInt(A)
B = parseInt(B)

const resultSum = (A + B).toString()
const resultSub = (A - B).toString()
const resultMul = (A * B).toString()
const resultDiv = Math.floor(A / B).toString() // 몫만 출력한다.
const resultMod = (A % B).toString()

const resultArr = [resultSum, resultSub, resultMul, resultDiv, resultMod]

process.stdout.write(resultArr.join('\n'))
```

<br />

- #### 가독성과 추후 실수 방지를 위해서 습관을 들여놓자는 의미로 parseInt 를 했다.

- #### console.log 에서 발생하는 불필요한 개행이 나중에 문제되지 않도록 process 의 함수를 쓴다.

    + 연습용으로 써본다.

- #### 코드를 더 짧게 작성할 수는 있으나, 가독성 좋은 코드를 연습한다는 의미로 저렇게 작성했다.

### **Python**

``` python
import sys
import math

input = sys.stdin.readline
output = sys.stdout.write

A, B = map(int, input().split())

output(f"{A + B}\n")
output(f"{A - B}\n")
output(f"{A * B}\n")
output(f"{math.floor(A / B)}\n")
output(f"{A % B}\n")
```

## **설명**

- 3자리 자연수 A, B 가 한 줄씩 들어온다. (1 이상 10^12 이하)

- A * B * C 를 출력한다.

- C 나 Java 같은 경우는 int 범위를 넘는것을 고려하여야 한다.

</div>

<div class="block_grey">

## **소스 코드**

<div class="block_green_transparent">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().split(' ')
let [A, B, C] = inputs

A = parseInt(A)
B = parseInt(B)
C = parseInt(C)

const result = A + B + C

process.stdout.write(result.toString())
```

<br />

- #### 가독성과 추후 실수 방지를 위해서 습관을 들여놓자는 의미로 parseInt 를 했다.

- #### console.log 에서 발생하는 불필요한 개행이 나중에 문제되지 않도록 process 의 함수를 쓴다.

    + 연습용으로 써본다.

- #### 가독성 떨어지는 짧은 코드 대신에 일부러 길게 풀어서 썼다.

- #### Node 는 int 범위 벗어나면 알아서 변환해준다.

</div>

<div class="block_blue_transparent">

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

A, B, C = map(int, input().split())

result = A + B + C

output(f"{result}")
```

<br />

- #### 나중에 대량 입출력이 필요할때는 대비해 sys 모듈을 쓰는걸 습관화한다.

- #### Python 의 정수형은 알아서 끝없이 길어진다. 형변환이 필요없다.

## **설명**

- 입력받은 문자열 뒤에 ??! 를 붙여서 출력한다.

- 원래 문제 이름은 ??! 인데, 이를 파일명에 포함할 수 없어서 실제 파일에는 문제명을 적지 않았다.

</div>

<div class="block_grey">

## **소스 코드**

<div class="block_green_transparent">

### **Node**

``` js
const fs = require('fs')
const input = fs.readFileSync('/dev/stdin').toString().trim()

process.stdout.write(input + '??!')

```

<br />

- #### 가독성과 추후 실수 방지를 위해서 습관을 들여놓자는 의미로 parseInt 를 했다.

- #### console.log 에서 발생하는 불필요한 개행이 나중에 문제되지 않도록 process 의 함수를 쓴다.

    + 연습용으로 써본다.
    
- #### 입력시 맨 뒤에 개행문자가 들어오기 때문에 trim 을 해준다.

</div>

<div class="block_blue_transparent">

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

inputString = input().strip()

output(f"{inputString + '??!'}")
```

<br />

- #### 입력시 맨 뒤에 개행문자가 들어오기 때문에 strip 을 해준다.

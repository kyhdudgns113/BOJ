## **설명**

- 2 이상 10,000 이하 정수 3개 A, B, C 를 받는다.

- 첫째 줄에 (A+B)%C, 둘째 줄에 ((A%C) + (B%C))%C, 셋째 줄에 (A×B)%C, 넷째 줄에 ((A%C) × (B%C))%C를 출력한다.

</div>

<div class="block_grey">

## **소스 코드**

<div class="block_green_transparent">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().split(' ')

let [A, B, C] = inputs

A = parseInt(A)
B = parseInt(B)
C = parseInt(C)

const resultsArr = [
  (A + B) % C,
  ((A % C) + (B % C)) % C,
  (A * B) % C,
  ((A % C) * (B % C)) % C
]

process.stdout.write(resultsArr.join('\n'))
```

<br />

- #### 가독성과 추후 실수 방지를 위해서 습관을 들여놓자는 의미로 parseInt 를 했다.

- #### console.log 에서 발생하는 불필요한 개행이 나중에 문제되지 않도록 process 의 함수를 쓴다.

    + 연습용으로 써본다.

- #### 결과값이 뭔지 코드에 표시하기 위해 굳이 const resultsArr 구문을 사용했다.

</div>

<div class="block_blue_transparent">

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

A, B, C = map(int, input().split(' '))

output(f"{(A + B) % C}\n")
output(f"{((A % C) + (B % C)) % C}\n")
output(f"{(A * B) % C}\n")
output(f"{((A % C) * (B % C)) % C}")
```

<br />

- #### 나중에 대량 입출력이 필요할때는 대비해 sys 모듈을 쓰는걸 습관화한다.

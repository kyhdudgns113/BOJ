## **설명**

- 3자리 자연수 A, B 가 한 줄씩 들어온다.

- A * B 를 사람이 계산하는 과정에서 나오는 수를 차례대로 한 줄씩 출력한다.

</div>

<div class="block_grey">

## **소스 코드**

<div class="block_green_transparent">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().split('\n')

let [num1, num2] = inputs

num1 = parseInt(num1)
num2 = parseInt(num2)

const num2_1 = num2 % 10
const num2_10 = Math.floor(num2 / 10) % 10
const num2_100 = Math.floor(num2 / 100)

// prettier-ignore
const resultArr = [
  num1 * num2_1,
  num1 * num2_10,
  num1 * num2_100,
  num1 * num2
]

process.stdout.write(resultArr.join('\n'))
```

<br />

- #### 가독성과 추후 실수 방지를 위해서 습관을 들여놓자는 의미로 parseInt 를 했다.

- #### console.log 에서 발생하는 불필요한 개행이 나중에 문제되지 않도록 process 의 함수를 쓴다.

    + 연습용으로 써본다.

- #### 가독성 떨어지는 짧은 코드 대신에 일부러 길게 풀어서 썼다.

- #### 결과값이 뭔지 코드에 표시하기 위해 굳이 const resultsArr 구문을 사용했다.

### **Python**

``` python
import sys
import math

input = sys.stdin.readline
output = sys.stdout.write

num1 = int(input())
num2 = int(input())

num2_1 = num2 % 10
num2_10 = (num2 // 10) % 10
num2_100 = num2 // 100

output(f"{num1 * num2_1}\n")
output(f"{num1 * num2_10}\n")
output(f"{num1 * num2_100}\n")
output(f"{num1 * num2}\n")
```

<br />

- #### 나중에 대량 입출력이 필요할때는 대비해 sys 모듈을 쓰는걸 습관화한다.

## **소스 코드**

<div class="block_green_transparent">

### **Node**

``` js
const fs = require('fs')
const y = fs.readFileSync('/dev/stdin').toString()

const result = parseInt(y) - 543

process.stdout.write(result.toString())


```

<br />

- #### 가독성과 추후 실수 방지를 위해서 습관을 들여놓자는 의미로 parseInt 를 했다.

- #### console.log 에서 발생하는 불필요한 개행이 나중에 문제되지 않도록 process 의 함수를 쓴다.

    + 연습용으로 써본다.

- #### 결과값이 뭔지 코드에 표시하기 위해 굳이 const result 구문을 사용했다.

</div>

<div class="block_blue_transparent">

### **Python**

``` python
import sys

input = sys.stdin.readline
output = std.stdout.write

y = int(input())

result = y - 543

output(f"{result}")
```

<br />

- #### 결과값이 뭔지 굳이 표시를 해주기 위해 result = y - 543 코드를 작성했다.

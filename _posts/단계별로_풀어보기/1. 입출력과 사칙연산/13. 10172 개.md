## **설명**

- 고양이 형태의 텍스트를 출력하면 된다.

- \ 과 ' 를 어떻게 잘 출력하는지가 중요하다.

- 각각 정규식, 문자열 시작과 끝을 표현할때 쓰인다.

</div>

<div class="block_grey">

## **소스 코드**

<div class="block_green_transparent">

### **Node**

``` js
console.log('|\\_/|')
console.log('|q p|   /}')
console.log('( 0 )"""\\')
console.log('|"^"`    |')
console.log('||_/=\\\\__|')
```

<br />

- #### ` 를 쓰면 ' 를 직접 써도 출력된다.

- #### \ 는 정규식 표현이나 \n 등의 구분자? 로 쓰이기도 하여 두 번 써줘야 한다.

- #### 여기서는 그냥 console.log 썼다.

</div>

<div class="block_blue_transparent">

### **Python**

``` python
print('|\\_/|')
print('|q p|   /}')
print('( 0 )"""\\')
print('|"^"`    |')
print('||_/=\\\\__|')
```

<br />

- #### 파이썬은 ' 나 \ 앞에 \ 를 붙여줘야 그대로 출력된다.

- #### 여기서는 편의상 print 함수로 출력했다.

<div class="block_white">

## **설명**

- 10 미만의 두 자연수 A, B 를 입력받고 A * B 를 출력한다.

</div>

<div class="block_grey">

## **소스 코드**

<div class="block_green_transparent">

### **Node**

``` js
const fs = require('fs')
const input = fs.readFileSync('/dev/stdin').toString().split(' ')
let [A, B] = input

A = parseInt(A)
B = parseInt(B)

let result = A * B
result = result.toString()

process.stdout.write(result)
```

<br />

- #### 가독성과 추후 실수 방지를 위해서 습관을 들여놓자는 의미로 parseInt 를 했다.

- #### console.log 에서 발생하는 불필요한 개행이 나중에 문제되지 않도록 process 의 함수를 쓴다.

    + 연습용으로 써본다.

</div>

<div class="block_blue_transparent">

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

A, B = map(int, input().split())

print(f"{A * B}")
```

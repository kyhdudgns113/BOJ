## **설명**

- 두 정수 A, B 를 입력받고, A + B 를 출력하는 문제

- 이 역시 입출력에 익숙해지기 위해 손풀기로 풀어본다.

- <s>블로그 게시글 작성하는게 훨씬 오래 걸린다.</s>

</div>

<div class="block_grey">

## **소스 코드**

<div class="block_green_transparent">

### **Node**

``` js
const fs = require("fs");
const input = fs.readFileSync("/dev/stdin").toString().split(" ");
const a = parseInt(input[0]);
const b = parseInt(input[1]);

process.stdout.write((a + b).toString())
```

<br />

#### fs.readFileSync 가 입력 방식중에서 제일 빠르다.

- readline 함수는 매 줄마다 입력을 수행한다.

- 입력 줄이 많아지면 비효울적이 된다.

</div>

<div class="block_orange_transparent">

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

A, B = map(int, input().split())

output(f"{A + B}")
```

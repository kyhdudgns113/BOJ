## **소스 코드**

<div class="block_green">

### **Node**

``` js
// 입력
const fs = require('fs')
const input = fs.readFileSync(0, 'utf-8').toString().trim().split('\n')[0]

// 알파벳별 값 매핑
const numMap = {
  'A': 3, 'B': 3, 'C': 3,
  'D': 4, 'E': 4, 'F': 4,
  'G': 5, 'H': 5, 'I': 5,
  'J': 6, 'K': 6, 'L': 6,
  'M': 7, 'N': 7, 'O': 7,
  'P': 8, 'Q': 8, 'R': 8, 'S': 8,
  'T': 9, 'U': 9, 'V': 9,
  'W': 10, 'X': 10, 'Y': 10, 'Z': 10
}

// 문자열을 char 배열로 변환
const word = input.split('')

// 결과갑 생성 및 출력
const result = word.reduce((acc, char) => (acc += numMap[char]), 0)

process.stdout.write(`${result}`)

```

### **공통**

- #### 리눅스 환경에서 이상한 문자가 추가되어 문자열에 들어가는것을 방지하고자 trim 을 썼다.

- #### 대용량 입출력을 연습하기 위해 process.stdout.write 함수를 쓰는 습관을 들인다.

</div>

- #### 단어의 알파벳마다 숫자들을 더한다

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

word = input().strip()

num_map = {
    'A': 3, 'B': 3, 'C': 3,
    'D': 4, 'E': 4, 'F': 4,
    'G': 5, 'H': 5, 'I': 5,
    'J': 6, 'K': 6, 'L': 6,
    'M': 7, 'N': 7, 'O': 7,
    'P': 8, 'Q': 8, 'R': 8, 'S': 8,
    'T': 9, 'U': 9, 'V': 9,
    'W': 10, 'X': 10, 'Y': 10, 'Z': 10
}

result = sum(num_map[char] for char in word)

output(f"{result}")
```

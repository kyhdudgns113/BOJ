## **소스 코드**

<div class="block_green">

### **Node**

``` js
// 입력
const fs = require('fs')
const inputArr = fs.readFileSync('/dev/stdin').toString().split('\n')

// 입력 parsing
const N = +inputArr[0] // unused
const numString = inputArr[1]

// 입력받은 문자열을 바이트 배열로 변환
const encoder = new TextEncoder()
const byteArr = encoder.encode(numString)
const baseVal = encoder.encode('0')

// 결과값 계산
const result = byteArr.reduce((acc, val) => (acc += val - baseVal), 0)

// 결과값 출력
process.stdout.write(`${result}`)
```

### **공통**

- #### 리눅스 환경에서 이상한 문자가 추가되어 문자열에 들어가는것을 방지하고자 trim 을 썼다.

- #### 대용량 입출력을 연습하기 위해 process.stdout.write 함수를 쓰는 습관을 들인다.

</div>

<div class="block_grey tp_80">

### **특이점**

- #### 숫자들이 담긴 문자열을 ByteArr 로 변환하여 아스키코드의 배열로 만들었다.

- #### 숫자들을 더해야 하므로 val - "0의 아스키코드" 를 누적합 연산에 사용했다.

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

N = int(input())
numArr = map(int, list(input().strip()))

output(f"{sum(numArr)}")
```

<div class="block_white tp_80">

### **공통**

- #### Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.

- #### output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.

<div class="block_grey tp_80">

### **특이점**

- #### Python 은 map 함수로 문자열을 편하게 숫자의 배열로 만들 수 있다.

- #### 이후 sum 함수로 그들의 합을 구하면 편리하다.

- #### **.strip() 안하니까 타입에러 뜬다**

## **설명**

### **입력**

- S: 문자열

- i: 인덱스

</div>

<div class="block_green">

### **출력**

- 문자열 S 에서 i 번째 문자 (i 는 1부터)

</div>

</div> <!-- 설명 -->

<div class="block_grey">

## **소스 코드**

<div class="block_green">

### **Node**

``` js
const fs = require('fs')
const input = fs.readFileSync('/dev/stdin').toString().trim().split('\n')

const S = input[0]
const i = parseInt(input[1])

process.stdout.write(S[i - 1])
```

<br />

- #### 리눅스 환경에서 이상한 문자가 추가되어 문자열에 들어가는것을 방지하고자 trim 을 썼다.

- #### 대용량 입출력을 연습하기 위해 process.stdout.write 함수를 쓰는 습관을 들인다.

</div>

<div class="block_blue">

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

S = list(input().strip())
i = int(input())

output(f"{S[i - 1]}")
```

<br />

- #### 개행문자등을 제거하기위해 S 에서는 trim 을 썼다.

- #### Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.

- #### output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

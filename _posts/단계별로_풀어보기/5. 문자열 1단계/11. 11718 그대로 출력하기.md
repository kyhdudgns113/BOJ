## **소스 코드**

<div class="block_green">

### **Node**

``` js
// 입력
const fs = require('fs')
const inputs = fs.readFileSync(0, 'utf-8').toString().trim().split('\n')

// 출력할 문자열, 출력함수 자주 쓰면 오래걸려서 미리 이거 쓰는 훈련
let result = ''

inputs.forEach(strs => (result += strs + '\n'))

process.stdout.write(result)
```

<br />

<div class="block_white tp_80">

### **공통**

- #### 리눅스 환경에서 이상한 문자가 추가되어 문자열에 들어가는것을 방지하고자 trim 을 썼다.

- #### 대용량 입출력을 연습하기 위해 process.stdout.write 함수를 쓰는 습관을 들인다.

</div>

  output(S)

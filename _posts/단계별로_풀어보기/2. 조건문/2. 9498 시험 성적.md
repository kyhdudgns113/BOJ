## **설명**

- 점수를 입력받는다.

- 90점 이상이면 A, 10점 단위로 B, C, D, 그 이하는 F 를 출력한다.

## **소스 코드**

<div class="block_green_transparent">

### **Node (If)**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString()
const score = parseInt(inputs)

if (score >= 90) {
  process.stdout.write('A')
} // ::
else if (score >= 80) {
  process.stdout.write('B')
} // ::
else if (score >= 70) {
  process.stdout.write('C')
} // ::
else if (score >= 60) {
  process.stdout.write('D')
} // ::
else {
  process.stdout.write('F')
}
```

<br />

- #### 정석적으로 if 문을 사용했다.

- #### 나중에 대량 입출력이 필요할때를 대비하여 저런 입출력을 사용한다.

- #### 충분히 더 짧은 코드를 작성할 수 있으나, 가독성등을 위해 저렇게 짰다.

    + 어차피 실제로 작업할때는 잘 쓰지 않는다.

    + 불필요한 잔기술인 경우가 종종 있다.

    + <s>옛날에 이미 많이 해봤다</s>

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

score = int(input())

scoreDiv10 = score // 10

if scoreDiv10 >= 9:
  output(f"A")
elif scoreDiv10 >= 8:
  output(f"B")
elif scoreDiv10 >= 7:
  output(f"C")
elif scoreDiv10 >= 6:
  output(f"D")
else:
  output(f"F")
```

<br />

- #### if 문 코드만 작성했다.

- #### Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.

- #### output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

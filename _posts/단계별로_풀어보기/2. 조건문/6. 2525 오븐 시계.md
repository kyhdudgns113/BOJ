## **설명**

- 첫째줄에 A, B 를 입력받는다.

- 둘째줄에 C 를 입력받는다.

- A 시 B 분에 시작해서 C 분 뒤를 출력한다.

</div>

<div class="block_grey">

## **소스 코드**

<div class="block_green_transparent">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().split('\n')
const [a, b] = inputs[0].split(' ')
const c = inputs[1]

const [A, B, C] = [parseInt(a), parseInt(b), parseInt(c)]

const ABC = 60 * A + B + C

const hour = Math.floor(ABC / 60) % 24
const minute = ABC % 60

process.stdout.write(`${hour} ${minute}`)
```

<br />

- #### 지저분한 조건문 대신에 수학으로 풀었다.

- #### 나중에 대량 입출력이 필요할때를 대비하여 저런 입출력을 사용한다.

- #### 충분히 더 짧은 코드를 작성할 수 있으나, 가독성을 위해 저렇게 짰다.

    + 어차피 실제로 작업할때는 잘 쓰지 않는다.

    + 불필요한 잔기술인 경우가 종종 있다.

    + <s>옛날에 이미 많이 해봤다</s>

</div>

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

A, B = map(int, input().split())
C = int(input())

ABC = 60 * A + B + C

hour = (ABC // 60) % 24
minute = ABC % 60

output(f"{hour} {minute}")
```

<br />

- #### Python 도 수학으로 풀었다.

- #### Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.

- #### output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

- 해당 좌표가 몇 사분면에 있는지 정수형태로 출력한다.

## **소스 코드**

<div class="block_green_transparent">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync(0, 'utf-8').toString().split('\n')
// const inputs = fs.readFileSync('/dev/stdin').toString().split('\n')

const [a, b] = inputs

const A = parseInt(a)
const B = parseInt(b)

let result = ''

if (A > 0) {
  if (B > 0) {
    result = '1'
  } // ::
  else {
    result = '4'
  }
} // ::
else {
  if (B > 0) {
    result = '2'
  } // ::
  else {
    result = '3'
  }
}

process.stdout.write(result)
```

<br />

- #### Node 는 정석적으로 조건문을 써서 풀었다.

- #### 이 문제에서는 dev stdin 쓰면 권한 없다는 오류가 뜬다.

    + 저렇게 바꿔주라고 백준 홈페이지에 나와있었다.

- #### 나중에 대량 입출력이 필요할때를 대비하여 저런 입출력을 사용한다.

- #### 충분히 더 짧은 코드를 작성할 수 있으나, 가독성을 위해 저렇게 짰다.

    + 어차피 실제로 작업할때는 잘 쓰지 않는다.

    + 불필요한 잔기술인 경우가 종종 있다.

    + <s>옛날에 이미 많이 해봤다</s>

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

X = int(input())
Y = int(input())

xx = 1.5 if X > 0 else 0.5
yy = -1 if Y > 0 else 1

result = int(2.5 + xx * yy)

output(f"{result}")
```

<br />

- #### Python 은 수학으로 풀었다.

- #### X 가 양수면 사분면의 값은 2.5 를 기준으로 1.5씩 변한다.

- #### X 가 음수면 사분면의 값은 2.5 를 기준으로 0.5씩 변한다.

- #### Y 가 양수면 그 값을 빼고, 양수면 그 값을 더하면 사분면의 값이 나온다.

- #### Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.

- #### output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

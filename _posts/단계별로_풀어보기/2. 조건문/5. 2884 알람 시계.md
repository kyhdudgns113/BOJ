## **설명**

- 시간 H, 분 M 을 한 줄로 입력받는다.

- 해당 시간에서 45분 이전의 시간을 출력한다.

</div>

<div class="block_grey">

## **소스 코드**

<div class="block_green_transparent">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().split(' ')

const [h, m] = inputs

const [H, M] = [parseInt(h), parseInt(m)]

const sumHM = 60 * H + M + 1440
const resultHM = sumHM - 45

const resultH = Math.floor(resultHM / 60) % 24
const resultM = resultHM % 60

process.stdout.write(resultH.toString() + ' ' + resultM.toString())
```

<br />

- #### 지저분한 조건문 대신에 수학으로 풀었다.

- #### 1시간을 60분으로 환산한다.

- #### 목표 시각의 분에서 45를 빼고, 그것의 시, 분 값을 구하고 출력한다.

- #### 이 문제에서는 dev stdin 쓰면 권한 없다는 오류가 뜬다.

    + 저렇게 바꿔주라고 백준 홈페이지에 나와있었다.

- #### 나중에 대량 입출력이 필요할때를 대비하여 저런 입출력을 사용한다.

- #### 충분히 더 짧은 코드를 작성할 수 있으나, 가독성을 위해 저렇게 짰다.

    + 어차피 실제로 작업할때는 잘 쓰지 않는다.

    + 불필요한 잔기술인 경우가 종종 있다.

    + <s>옛날에 이미 많이 해봤다</s>

</div>

### **Python**

``` python
import sys, math

input = sys.stdin.readline
output = sys.stdout.write

H, M = map(int, input().split())

HM = 60 * H + M + 1440 - 45

resultH = (HM // 60) % 24
resultM = HM % 60

output(f"{resultH} {resultM}")
```

<br />

- #### Python 도 수학으로 풀었다.

- #### Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.

- #### output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

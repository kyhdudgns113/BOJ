## **설명**

- 연도를 입력받는다.

- 4로 나누어 떨어지지만 100으로는 나누어 떨어지지 않으면 윤년이다.

- 400으로 나누어 떨어지면 윤년이다.

</div>

<div class="block_grey">

## **소스 코드**

<div class="block_green_transparent">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString()

const year = parseInt(inputs)

const isFour = year % 4 === 0
const isHundred = year % 100 === 0
const isFHundred = year % 400 === 0

const isYunYear = (isFour && !isHundred) || isFHundred

const result = isYunYear ? '1' : '0'

process.stdout.write(result)
```

<br />

- #### 각 조건에 대한 변수를 **굳이** 변수로 만들었다.

    + 실제로 작업할때는 저렇게 코드를 짜는게 유지보수에 도움이 된다.

- #### 나중에 대량 입출력이 필요할때를 대비하여 저런 입출력을 사용한다.

- #### 충분히 더 짧은 코드를 작성할 수 있으나, 가독성등을 위해 저렇게 짰다.

    + 어차피 실제로 작업할때는 잘 쓰지 않는다.

    + 불필요한 잔기술인 경우가 종종 있다.

    + <s>옛날에 이미 많이 해봤다</s>

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

year = int(input())

if year % 4 == 0 and year % 100 != 0 or year % 400 == 0:
  output(f"1")
else:
  output(f"0")
```

<br />

- #### 이건 그냥 짧게 작성했다.

- #### Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.

- #### output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

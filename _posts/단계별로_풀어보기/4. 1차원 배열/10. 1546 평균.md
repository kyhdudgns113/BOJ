## **설명**

<div class="block_angle_60">

### **입력**

- N : 시험 치른 과목 수

- scoreArr[] : 한 줄로 들어오는 N 개 과목의 점수

</div>

<div class="block_angle_90">

### **상황**

- 원래 점수를 가장 잘 맞은 과목 점수에 비례한 값으로 바꾸었다.

- 원래 점수가 40점, 가장 잘 본 과목 점수가 80점이면 40/80 * 100 인 50점으로 만들었다.

</div>

<div class="block_angle_120">

### **출력**

- 바뀐 점수들의 평균

</div>

## **소스 코드**

<div class="block_green">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().split('\n')

const N = +inputs[0]

let sums = 0
let maxNum = 0

inputs[1].trim().split(' ').forEach(val => {
  const numb = +val
  sums += numb
  maxNum = Math.max(maxNum, numb)
})

process.stdout.write(`${((sums / maxNum) * 100) / N}`)
```

<br />

- #### **딱히 특별할게 없는 연산이다**

- #### **대용량 입출력을 연습하기 위해 process.stdout.write 함수를 쓰는 습관을 들인다.**

</div>

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

N = int(input())

scoreArr = list(map(int, input().split()))

output(f"{sum(scoreArr) / max(scoreArr) * 100 / N}")
```

<br />

- #### **Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.**

- #### **output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.**

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

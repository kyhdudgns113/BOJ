## **설명**

<div class="block_angle_60">

### **입력**

- N, M : 100 이하의 두 자연수

- (i, j)[] : M 줄에 걸쳐 들어오는 100 이하의 자연수들

</div>

<div class="block_angle_90">

### **상황**

- 1번부터 N번까지의 바구니가 있다.

- i, j 가 들어오면 i 번째부터 j 번째까지 바구니를 역순으로 만든다.

</div>

<div class="block_angle_120">

### **출력**

- 바구니들의 번호를 출력한다.

</div>

## **소스 코드**

<div class="block_green">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().split('\n')

const [N, M] = inputs[0].split(' ').map(Number)

const resultArr = Array(N)
  .fill(0)
  .map((_, idx) => idx + 1)

for (let idx = 0; idx < M; idx++) {
  const [i, j] = inputs[idx + 1].split(' ').map(Number)

  const partArr = resultArr.slice(i - 1, j)
  partArr.reverse()
  resultArr.splice(i - 1, j - i + 1, ...partArr)
}

process.stdout.write(`${resultArr.join(' ')}`)
```

<br />

- #### **i, j 가 들어오면 해당 범위의 배열을 복사한다.**

- #### **복사한 배열을 reverse 함수로 뒤집는다.**

- #### **기존 배열을 splice 함수로 일부를 잘라내고 뒤집은 배열로 채워넣는다.**

- #### 연산량은 O(M*N) 이다.

- #### **대용량 입출력을 연습하기 위해 process.stdout.write 함수를 쓰는 습관을 들인다.**

</div>

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

N, M = map(int, input().split())

resultArr = [i + 1 for i in range(N)]

for m in range(M):
  i, j = map(int, input().split())
  resultArr[i - 1: j] = resultArr[i - 1:j][::-1]

output(" ".join(str(r) for r in resultArr))
```

<br />

- #### **Node 랑 알고리즘이 같으며, 연산량도 O(M*N)으로 같다**

- #### **Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.**

- #### **output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.**

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

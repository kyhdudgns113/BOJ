### **입력**

- 1,000 이하의 자연수 10개, 1줄씩

<div class="block_angle_120">

### **출력**

- 수들을 42로 나누었을때의 나머지의 갯수를 출력

</div>

</div> <!-- 설명 -->

<div class="block_grey">

## **소스 코드**

<div class="block_green">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().trim().split('\n').map(Number)

const dict = {}

inputs.forEach(num => (dict[num % 42] = 1))

process.stdout.write(`${Object.keys(dict).length}`)
```

<br />

- #### **dictionary 에 나머지 값을 넣고 그 갯수를 출력하면 된다.**

- #### **연산량은 O(log n) 이 된다.

- #### **대용량 입출력을 연습하기 위해 process.stdout.write 함수를 쓰는 습관을 들인다.**

</div>

<div class="block_blue">

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

nums = [int(input()) for i in range(10)]

results = [0 for i in range(42)]

for num in nums:
  results[num % 42] = 1

output(f"{results.count(1)}")
```

<br />

- #### **python 은 나머지 배열을 만들고, 나머지 값이 존재하면 1로 채워넣었다.**

- #### **이후, 배열에서 1의 갯수를 찾으면 된다.**

- #### **연산량은 O(n) 이다.**

- #### **Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.**

- #### **output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.**

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

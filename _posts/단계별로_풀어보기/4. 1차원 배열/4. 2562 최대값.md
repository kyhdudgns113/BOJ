## **소스 코드**

<div class="block_green">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().split('\n')

const numArr = inputs.map(Number)

const maxNumber = Math.max(...numArr)
const maxNumIdx = numArr.indexOf(maxNumber) + 1

process.stdout.write(`${maxNumber}\n${maxNumIdx}`)
```

<br />

- #### **for 문을 쓰는게 더 빠르다.**

    + for 문으로 풀면 반복문을 하나만 돌아도됨

- #### **직관성을 더 강조하느라 저렇게 코드를 짰다.**

    + 어차피 시간, 메모리에 문제가 없다면 저렇게 풀어도 무방하긴 하다.

- #### **대용량 입출력을 연습하기 위해 process.stdout.write 함수를 쓰는 습관을 들인다.**

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

numArr = list(int(input()) for i in range(9))

maxNum = max(numArr)
maxNumIdx = numArr.index(maxNum) + 1

output(f"{maxNum}\n{maxNumIdx}")
```

<br />

- #### **Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.**

- #### **output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.**

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

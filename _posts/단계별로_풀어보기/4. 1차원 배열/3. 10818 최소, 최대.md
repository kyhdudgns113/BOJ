## **소스 코드**

<div class="block_green">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().split('\n')

const N = +inputs[0]
const numArr = inputs[1].split(' ').map(Number)

process.stdout.write(`${Math.min(...numArr)} ${Math.max(...numArr)}`)
```

<br />

- #### for 문으로 직접 찾는 방법도 있다.

- #### min 함수와 max 함수를 쓰는게 훨씬 직관적이다.

- #### 대용량 입출력을 연습하기 위해 process.stdout.write 함수를 쓰는 습관을 들인다.

</div>

<div class="block_blue">

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

N = int(input())
numArr = list(map(int, input().split()))

output(f"{min(numArr)} {max(numArr)}")
```

<br />

- #### Python 도 min, max 함수를 썼다.

- #### Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.

- #### output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

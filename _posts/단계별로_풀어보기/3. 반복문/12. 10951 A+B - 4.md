## **소스 코드**

<div class="block_green">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().split('\n')

inputs.forEach(row => {
  if (row.length > 0) {
    const arrs = row.split(' ')
    const [A, B] = [+arrs[0], +arrs[1]]
    process.stdout.write(`${A + B}\n`)
  }
})
```

<br />

- #### 메모리 제한이 256MB 라서 입력을 전부 저장해도 상관 없다.

    + 빠른 코드작성, 속도를 위해서 메모리를 희생한다.

- #### 대용량 입출력을 연습하기 위해 process.stdout.write 함수를 쓰는 습관을 들인다.

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

while True:
  try:
    A, B = map(int, input().split())
    output(f"{A + B}\n")
  except:
    exit()
```

<br />

- #### 입력받을게 없는데 입력함수를 쓰면 에러가 발생한다.

    + 이걸 catch 했을때 종료하면 된다.

- #### Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.

- #### output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

### **C**

``` c
#include <stdio.h>

int main(int argc, char **argv) {
  int a = 0, b = 0;

  while(scanf("%d %d", &a, &b) != -1) {
    printf("%d\n", a + b);
  }

  return 0
}
```

<br />

- #### scanf 에서 에러가 나면 -1 을 리턴한다.

- #### 이때까지 입력을 받으면 된다.

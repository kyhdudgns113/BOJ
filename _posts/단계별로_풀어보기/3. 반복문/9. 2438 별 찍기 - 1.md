- N 개의 줄에 대해서 i 번쨰줄에 별 i 개 찍기

</div>

<div class="block_grey">

## **소스 코드**

<div class="block_green">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().split('\n')[0]

const N = +inputs

for (let i = 1; i <= N; i++) {
  process.stdout.write('*'.repeat(i) + '\n')
}
```

<br />

- #### 대용량 입출력을 연습하기 위해 process.stdout.write 함수를 쓰는 습관을 들인다.

</div>

<div class="block_blue">

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

N = int(input())

output('\n'.join('*' * i for i in range(1, N + 1)))
```

<br />

- #### JavaScript 랑 join 쓰는게 정반대인걸 배웠다.

- #### Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.

- #### output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

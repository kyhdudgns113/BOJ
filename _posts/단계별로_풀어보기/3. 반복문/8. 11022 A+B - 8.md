## **소스 코드**

<div class="block_green">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().split('\n')

const T = +inputs[0]

for (let i = 0; i < T; i++) {
  const inputRow = inputs[i + 1].split(' ')
  const [A, B] = [+inputRow[0], +inputRow[1]]
  process.stdout.write(`Case #${i + 1}: ${A} + ${B} = ${A + B}\n`)
}
```

<br />

- #### parseInt 보다 +를 붙이는게 속도가 빠르다는걸 알게됬다.

- #### 7번째 문제 코드를 복사해서 형식만 바꿨다 <s>개꿀</s>

- #### 대용량 입출력을 연습하기 위해 process.stdout.write 함수를 쓰는 습관을 들인다.

</div>

<div class="block_blue">

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

T = int(input())

for i in range(1, T + 1):
  A, B = map(int, input().split())
  output(f"Case #{i}: {A} + {B} = {A + B}\n")
```

<br />

- #### Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.

- #### output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

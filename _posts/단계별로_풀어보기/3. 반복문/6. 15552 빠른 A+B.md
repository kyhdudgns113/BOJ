## **설명**

<div class="block_yellow">

### **입력**

- 테스트 케이스 T (T <= 1,000,000)

- T 줄에 걸쳐서 1,000 이하의 두 자연수 A, B

</div>

<div class="block_green">

### **출력**

- 각 테스트 케이스마다 A + B 의 값을 한 줄씩 출력

</div>

</div> <!-- 설명 -->

<div class="block_grey">

## **소스 코드**

<div class="block_green">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().split('\n')

const T = parseInt(inputs[0])
const resultArr = []

for (let i = 0; i < T; i++) {
  const [a, b] = inputs[i + 1].split(' ').map(Number)
  // const [a, b] = inputs[i + 1].split(' ').map(val => +val)
  resultArr.push(a + b)
}

process.stdout.write(resultArr.join('\n'))
```

<br />

- #### 테스트 케이스마다 출력함수를 쓰면 시간초과가 난다.

- #### 메모리가 충분하기에 결과값을 담아두고 한번에 출력했다.

</div>

<div class="block_blue">

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

T = int(input())

while T > 0:
  A, B = map(int, input().split())
  output(f"{A + B}\n")
  T -= 1
```

<br />

- #### 이미 예전부터 많이 쓰던 입출력 방식이라 수정할 필요는 없었다.

- #### Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.

- #### output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

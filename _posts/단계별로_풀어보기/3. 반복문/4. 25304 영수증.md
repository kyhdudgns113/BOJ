## **설명**

- 예상 금액 X 를 입력받는다.

- 산 품목의 종류 N 을 입력받는다.

- 이후 N 줄에 걸쳐 1개당 가격 A, 갯수 B 를 입력받는다.

- 예상 금액과 물건 가격 총합이 맞으면 Yes, 아니면 No 를 출력한다.

</div>

## **소스 코드**

<div class="block_green">

### **Node (정석 반복문)**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().split('\n')

const X = parseInt(inputs[0])
const N = parseInt(inputs[1])

let sums = 0

for (let i = 0; i < N; i++) {
  const inputRow = inputs[i + 2].split(' ')
  const [a, b] = [parseInt(inputRow[0]), parseInt(inputRow[1])]

  sums += a * b
}

if (X === sums) {
  process.stdout.write(`Yes`)
}
else {
  process.stdout.write('No')
}
```

<br />

- #### **일부러 반복문으로 풀었다.**

- #### 다시 process.stdout 을 쓰기 시작했다.

- #### 충분히 더 짧은 코드를 작성할 수 있으나, 가독성등을 위해 저렇게 짰다.

    + 어차피 실제로 작업할때는 잘 쓰지 않는다.

    + 불필요한 잔기술인 경우가 종종 있다.

    + <s>옛날에 이미 많이 해봤다</s>

### **Node (코드 간소화)**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().split('\n')

const X = parseInt(inputs[0])
const N = parseInt(inputs[1])

const result = Array(N)
  .fill(0)
  .map((_, idx) => inputs[idx + 2].split(' ').map(val => parseInt(val)))
  .reduce((res, [a, b]) => res + a * b, 0)

process.stdout.write(`${X === result ? 'Yes' : 'No'}`)
```

<br />

- #### **스끼리를 썼다.**

### **Python**

``` python
import sys, math

input = sys.stdin.readline
output = sys.stdout.write

X = int(input())
N = int(input())

sums = sum(math.prod(map(int, input().split())) for i in range(N))

output(f"{'Yes' if X == sums else 'No'}")
```

<br />

- #### 파이썬도 일종의 스킬을 썼다.

- #### Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.

- #### output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

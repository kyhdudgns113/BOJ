## **설명**

- 테스트 케이스 T 를 입력받는다.

- 이후 T 줄에 걸쳐서 두 수 A, B 를 입력받는다.

- A + B 를 한 줄 씩 출력한다.

</div>

<div class="block_grey">

## **소스 코드**

<div class="block_green_transparent">

### **Node**

``` js
const fs = require('fs')
const inputs = fs.readFileSync('/dev/stdin').toString().split('\n')

const T = parseInt(inputs[0])

for (let i = 0; i < T; i++) {
  const inputRow = inputs[i + 1].split(' ')
  const [A, B] = [parseInt(inputRow[0]), parseInt(inputRow[1])]

  process.stdout.write(`${A + B}\n`)
}
```

<br />

- #### 다시 process.stdout 을 쓰기 시작했다.

- #### 충분히 더 짧은 코드를 작성할 수 있으나, 가독성등을 위해 저렇게 짰다.

    + 어차피 실제로 작업할때는 잘 쓰지 않는다.

    + 불필요한 잔기술인 경우가 종종 있다.

    + <s>옛날에 이미 많이 해봤다</s>

</div>

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

T = int(input())

while T:
  A, B = map(int, input().split())
  output(f"{A + B}\n")
  T -= 1
```

<br />

- #### Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.

- #### output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.

## **설명**

- 정수의 바이트 크기 N 을 입력받는다.

- 바이트 크기 / 4 만큼 long, 마지막엔 int 를 붙여서 출력한다.

    + 각각 공백 1칸 둔다.

</div>

<div class="block_grey">

## **소스 코드**

<div class="block_green">

### **Node**

``` js
const inputs = fs.readFileSync('/dev/stdin').toString()

const N = parseInt(inputs)
const Ndiv4 = Math.floor(N / 4)

process.stdout.write(`${'long '.repeat(Ndiv4)}int`)
```

<br />

- #### **일부러 반복문으로 풀 필요성을 크게 못 느꼈다.**

- #### 다시 process.stdout 을 쓰기 시작했다.

</div>

<div class="block_blue">

### **Python**

``` python
import sys

input = sys.stdin.readline
output = sys.stdout.write

N = int(input())

result = 'long ' * (N // 4) + 'int'

output(f"{result}")
```

<br />

- #### 파이썬도 굳이 for 나 while 문으로 풀지 않으려 했다.

- #### Python 역시 대량 입출력에 익숙해지기 위해 저 함수들을 쓴다.

- #### output 은 format 을 지정하여 커스터마이징이 용이하도록 한다.

    + C 스타일의 잔재가 안쓴지 수년이 지나도 남아있다.
